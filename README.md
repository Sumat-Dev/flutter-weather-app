## Setup

1. **Get a Weather API Key**
   - Sign up at [WeatherAPI.com](https://www.weatherapi.com/)
   - Get your free API key
   - Copy `.env.example` to `.env` and add your API key:

     ```env
     WEATHER_API_KEY="YOUR_API_KEY"
     ```

2. **Install Dependencies**

   ```bash
   flutter pub get
   ```

3. **Run the App**

   ```bash
   flutter run
   ```
